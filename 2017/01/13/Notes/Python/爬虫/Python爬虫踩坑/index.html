打开url

返回一个文件对象，然后可以进行类似于文件对象的操作。urlopen()返回的对象提供如下方法：

- `read(),readline(),readlines(),fileno(),close()` # 使用类似文件对象
- `info()` # 返回一个httplib.HTTPMessage对象，表示远程服务器返回的头信息
- `getcode()` 返回一个http状态码
- `geturl()` 返回请求的URL
