如果你理解掌握了map和reduce的用法，就再也不用写以前那样lowd的循环了。事实上，在项目中不使用循环还真的是个挑战。当你使用几次map和reduce以后，会注意到更多的使用模式并被它吸引。还有一些普遍的用法是从Array中filtering或者plucking数据

因为这些模式经常被使用，人们就把他们放进了单独的库中。一些比较出名的库有：

  • Ramda
  • Lodash
  • Underscore
